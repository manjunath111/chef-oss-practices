# Chef Infra Client

> [!WARNING]
> This page in the Chef Open Source Software Practices repository is currently undergoing content review.

## Project

Chef Infra Client — A powerful configuration management tool that brings infrastructure automation to scale, ensuring system state consistency across platforms.

### Project Scope

* Chef Infra Client is responsible for enforcing desired system configurations across servers.
* It handles system resource definitions, dependencies and executions of IaC recorded in cookbooks.
* Built in Ruby with native extensions; uses Ohai for system profiling.
