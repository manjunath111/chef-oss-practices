## Project Name
Chef Habitat — Application automation framework designed to build, deploy, and manage applications in a consistent and repeatable way across all environments.

### Project Scope
* Chef Habitat enables packaging, running, and managing applications as portable, self-contained artifacts.
* It provides an automated build system, runtime environment, and dynamic configuration capability.
* Built in Rust and supports multiple runtime and packaging targets, including Linux, Windows, and containerized environments.

## Sub Projects
* **Builder** — Habitat's package depot, build automation service, and artifact promotion system.
* **Hab CLI** — Command-line interface for building and managing Habitat packages.
* **Supervisor** — Runtime process manager that handles service lifecycle, configuration updates, and peer communication.
